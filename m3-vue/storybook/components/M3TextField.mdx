# Text fields

Text fields let users enter text into a UI

[Design Kit (Figma)](https://www.figma.com/community/file/1035203688168086460)

<Inline tag={'p'} is={defineComponent({
  setup () {
    const text = ref('')

    return () => h(M3TextField, {
      value: text.value,
      label: 'E-mail',
      'onUpdate:value': value => text.value = value,
    }, {
      'leading-icon': () => h(M3Icon, { name: 'mail' }),
    })
  },
})} />
